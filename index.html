🛠 LAPTOP SETUP (one time) ▼
1
Download & Install Ollama Go to ollama.com → Download for Windows → Install it (takes ~2 min)
2
Download the Vision AI Model Open Command Prompt and run:
ollama pull llava:7b
Downloads ~4GB once. 7B model works great on 16GB RAM.
3
Allow phone to connect (run this once) In Command Prompt, run:
set OLLAMA_ORIGINS=* && set OLLAMA_HOST=0.0.0.0 && ollama serve
Keep this window open while using the app.
4
Find your laptop's local IP address Open Command Prompt and run:
ipconfig
Look for IPv4 Address under Wi-Fi — e.g. 192.168.1.5
5
Phone & laptop on same Wi-Fi? Make sure your phone is connected to the same Wi-Fi network as your laptop.
⚡ Run Step 3 command every time you restart your laptop before using the app.
💡 Want a one-click shortcut? Save the Step 3 command as a .bat file on your desktop — double-click it to start the AI server anytime.
